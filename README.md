# OpenSDLC v1

OpenSDLC는 소프트웨어 개발 생명주기(SDLC)를 아티팩트 중심으로 운영하는 AI Software Factory 플랫폼이다. 각 단계의 산출물은 단순 기록물이 아니라 다음 단계 에이전트의 실행 지시서로 사용되며, `Validation Over Self-Assertion`, `Full Traceability`, `Strict Adherence`, `No-Shortcut Principle`, `Sequential Single-Agent Execution`을 핵심 운영 원칙으로 둔다.
